# BINARYFLIPPER

A Deriv market analysis tool that provides real-time predictions across all markets.

## Features
- Over/Under, Rise/Fall, and Even/Odd predictions
- Real-time tick data visualization via Deriv WebSocket API
- Digit Circle Analysis with frequency tracking
- Multi-market support

## Usage
Open `index.html` in your browser. No build step required — pure HTML/CSS/JS.
